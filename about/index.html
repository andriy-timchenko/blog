Про мій блог

Блог згенерований за допомогою gohugo — це один із найпопулярніших генераторів статичних сайтів із відкритим кодом.

На моєму десктопі з Ubuntu 22.04 він легко вскановлюється командою sudo apt install hugo ,

Hugo is a static site engine written in Go.

It makes use of a variety of open source projects including:

  • Cobra
  • Viper
  • J Walter Weatherman
  • Cast

Learn more and contribute on GitHub.
